# Sniplify Support

## About

Sniplify is a lightweight macOS menu bar app that helps you quickly search and insert text snippets.

---

## How to Use

1. Launch the app
2. The app icon will appear in the macOS menu bar (top-right)
3. Press **Option + Space (⌥ Space)** to open the snippet search
4. Type to search your snippets
5. Press **Enter** to insert into the active application

---

## Permissions

Sniplify may request **Accessibility permission** in order to insert the selected snippet into the currently active app.

* Sniplify **does not monitor or record your keystrokes**
* All data is stored **locally on your device**
* No personal data is uploaded

---

## Troubleshooting

### The shortcut does not work

* Make sure the app is running
* Check if another app is using the same shortcut

### Snippet is not inserted

* Ensure Accessibility permission is enabled:
  System Settings → Privacy & Security → Accessibility
